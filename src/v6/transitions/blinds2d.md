# Blinds 2D

## Description

Divides the image in vertical bars that blinds and fades out.

## Options

| Name | Type | Default |
|------|------|---------|
| cols | Numeric | 10 |
| tileDuration | Numeric | 800 |
| tileDelay | Numeric | 100 |
| easing | String | linear |
