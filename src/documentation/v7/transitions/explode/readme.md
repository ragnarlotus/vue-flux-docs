# Explode

## Description

The image starts to explode from the center to outside.

## Options

| Name | Type | Default |
|------|------|---------|
| rows | Numeric | 9 |
| cols | Numeric | 9 |
| tileDuration | Numeric | 300 |
| tileDelay | Numeric | 100 |
| easing | String | linear |
