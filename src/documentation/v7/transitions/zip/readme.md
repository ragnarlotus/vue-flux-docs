# Zip

## Description

Divides the image in bars and slides them up and down alternately like a zip.

## Options

| Name | Type | Default |
|------|------|---------|
| cols | Numeric | 10 |
| tileDuration | Numeric | 600 |
| tileDelay | Numeric | 80 |
| easing | String | ease-in |
