# Zip

## Description

Divides the image in bars and slides them up and down alternately like a zip.

[Demo](../../../demos/transitions/zip)

## Options

| Name | Type | Default |
|------|------|---------|
| cols | number | 10 |
| tileDuration | number | 600 |
| tileDelay | number | 80 |
| easing | string | ease-in |
