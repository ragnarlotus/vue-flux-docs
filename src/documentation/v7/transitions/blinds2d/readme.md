# Blinds 2D

## Description

Divides the image in vertical bars that blinds and fades out.

[Demo](../../../demos/transitions/blinds2d)

## Options

| Name         | Type   | Default |
| ------------ | ------ | ------- |
| cols         | number | 10      |
| tileDuration | number | 800     |
| tileDelay    | number | 100     |
| easing       | string | linear  |
