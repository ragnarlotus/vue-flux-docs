# Camera

## Description

From outside to inside the image is being cropped like in a camera lens.

[Demo](../../../demos/transitions/camera)

## Options

| Name | Type | Default |
|------|------|---------|
| totalDuration | number | 900 |
| backgroundColor | string | #111 |
| easing | string | cubic-bezier(0.385, 0, 0.795, 0.560) |
