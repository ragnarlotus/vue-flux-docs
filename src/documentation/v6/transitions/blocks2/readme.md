# Blocks 2

## Description

The image is split in blocks that shrink and fade out in wave from a corner to the opposite.

## Options

| Name | Type | Default |
|------|------|---------|
| rows | Numeric | 8 |
| cols | Numeric | 8 |
| tileDuration | Numeric | 800 |
| tileDelay | Numeric | 80 |
| easing | String | ease |
