# Blinds 3D

## Description

Divides the image in vertical bars that blinds 180 deg to form the next image.

## Options

| Name | Type | Default |
|------|------|---------|
| cols | number | 6 |
| tileDuration | number | 800 |
| tileDelay | number | 150 |
| easing | string | ease-out |
