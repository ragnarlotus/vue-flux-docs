# Blinds 3D

## Description

Divides the image in vertical bars that blinds 180 deg to form the next image.

## Options

| Name | Type | Default |
|------|------|---------|
| cols | Numeric | 6 |
| tileDuration | Numeric | 800 |
| tileDelay | Numeric | 150 |
| easing | String | ease-out |
