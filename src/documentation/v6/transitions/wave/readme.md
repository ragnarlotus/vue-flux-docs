# Wave

## Description

Makes the image 3D and divides it in slices that turn vertically to display the next image.

## Options

| Name | Type | Default |
|------|------|---------|
| cols | Number | 8 |
| tileDuration | Number | 900 |
| tileDelay | Number | 110 |
| sideColor | String | #333 |
| easing | String | cubic-bezier(0.3, -0.3, 0.735, 0.285) |
