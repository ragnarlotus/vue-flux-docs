# Waterfall

## Description

Divides the image in bars and drops them down in turns.

## Options

| Name | Type | Default |
|------|------|---------|
| cols | Number | 10 |
| tileDuration | Number | 600 |
| tileDelay | Number | 90 |
| easing | String | cubic-bezier(0.55, 0.055, 0.675, 0.19) |
