# Explode

## Description

The image starts to explode from the center to outside.

## Options

| Name | Type | Default |
|------|------|---------|
| rows | number | 9 |
| cols | number | 9 |
| tileDuration | number | 300 |
| tileDelay | number | 100 |
| easing | string | linear |
