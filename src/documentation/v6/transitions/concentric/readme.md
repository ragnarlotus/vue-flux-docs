# Concentric

## Description

A concentric effect is performed by rotating the image converted into circles.

## Options

| Name | Type | Default |
|------|------|---------|
| circles | number | 7 |
| tileDuration | number | 800 |
| tileDelay | number | 150 |
| easing | string | linear |
