# Concentric

## Description

A concentric effect is performed by rotating the image converted into circles.

## Options

| Name | Type | Default |
|------|------|---------|
| circles | Numeric | 7 |
| tileDuration | Numeric | 800 |
| tileDelay | Numeric | 150 |
| easing | String | linear |
