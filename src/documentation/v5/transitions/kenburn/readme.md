### Description

Fades, zoom and moves current image to next.

### Options

| Name          | Type    | Default                                  |
| ------------- | ------- | ---------------------------------------- |
| totalDuration | Numeric | 6000                                     |
| easing        | String  | cubic-bezier(0.600, 0.040, 0.780, 0.335) |
