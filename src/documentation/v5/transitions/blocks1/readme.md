### Description

The image is split in blocks that shrink and fade out randomly.

### Options

| Name         | Type    | Default |
| ------------ | ------- | ------- |
| rows         | number |         |
| cols         | number |         |
| tileDuration | number | 300     |
| tileDelay    | number | 1000    |
| easing       | String  | linear  |
