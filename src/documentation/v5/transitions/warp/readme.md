### Description

A concentric effect is performed by rotating the image converted into circles in alternate direction.

### Options

| Name         | Type    | Default |
| ------------ | ------- | ------- |
| circles      | Numeric |         |
| tileDuration | Numeric | 800     |
| tileDelay    | Numeric | 150     |
| easing       | String  | linear  |
