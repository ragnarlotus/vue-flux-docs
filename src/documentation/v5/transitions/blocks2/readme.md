### Description

The image is split in blocks that shrink and fade out in wave from a corner to the opposite.

### Options

| Name         | Type    | Default |
| ------------ | ------- | ------- |
| rows         | number |         |
| cols         | number |         |
| tileDuration | number | 800     |
| tileDelay    | number | 80      |
| easing       | String  | ease    |
