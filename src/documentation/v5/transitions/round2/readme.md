### Description

Panels start to round vertically revealing the next image in upper arrow form leaving trail.

### Options

| Name         | Type    | Default |
| ------------ | ------- | ------- |
| rows         | number |         |
| cols         | number |         |
| tileDuration | number | 800     |
| tileDelay    | number | 100     |
| easing       | String  | linear  |
