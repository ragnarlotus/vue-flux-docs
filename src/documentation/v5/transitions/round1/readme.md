### Description

The image is split in blocks that turn 180 deg horizontally to form next image.

### Options

| Name         | Type    | Default  |
| ------------ | ------- | -------- |
| rows         | Numeric |          |
| cols         | Numeric |          |
| tileDuration | Numeric | 800      |
| tileDelay    | Numeric | 150      |
| easing       | String  | ease-out |
