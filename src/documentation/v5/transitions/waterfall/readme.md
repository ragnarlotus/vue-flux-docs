### Description

Divides the image in bars and drops them down in turns.

### Options

| Name         | Type   | Default |
| ------------ | ------ | ------- |
| numCols      | number |         |
| tileDuration | number | 600     |
| tileDelay    | number | 80      |
| easing       | string | ease-in |
